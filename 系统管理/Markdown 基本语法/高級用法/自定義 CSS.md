如果你正在設計你個性化的 CSS，你可以通過使用開發者工具來獲得更多元素資訊，提高設計效率。使用開發者工具，1）在 Windows 或 Linux 下，使用快捷鍵 `Ctrl+Shift+I`；2）在 macOS 下，使用快捷鍵 `Cmd+Opt+I` 即可。

對於 CSS 的修改，Obsidian 是自動檢測並會立即應用的。你不需要重啟 Obsidian 來檢視修改效果。
